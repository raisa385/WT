Convolutional Neural Network

What is it?

Convolutional Neural Network (CNN) is a special type of deep learning model that processes data organized in a structured grid like images. It applies filters to input data to detect features like edges and textures.

Applications

CNN has become the standard for computer vision, being used in tasks such as image recognition, classification and object detection. It has it's applications in medical image analysis (especially in X-rays and MRIs) and biometrics.
